# Telegram Weather Bot

Четвертый проект. Web-интерфейс через api/telegram bot.

Он может показывать погоду и рисовать график температуры.  
Бот и все функции задеплоины на google cloud run и работают с api телеграма в режиме 24/7.

## Команды бота https://t.me/AlekseiSAbot

- `/start` - показывает список команд
- `/weather` - показывает текущую погоду (я беру данные с моей функции в google cloud)
- `/chart` - присылает картинку с графиком температуры (тоже из cloud)

## как устроено

бот работает через webhook. когда человек пишет команду в телеграм, телеграм шлет запрос в мою функцию (google run).  
функция смотрит что за команда и возвращает текст или картинку.  

В репозитории представлены коды функций но непосредственно через интерпретатор они не запускаются.
